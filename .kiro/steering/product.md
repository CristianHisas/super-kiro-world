# Product Overview

Super Kiro World is a browser-based 2D platformer game built for the AWS Re:Invent workshop. Players control the Kiro mascot through a side-scrolling level, collecting purple gems while avoiding enemies and hazards. The game features classic platformer mechanics with a modern Kiro brand aesthetic, designed to be completed in a 2-hour workshop session.

## Core Features
- Platform-based level progression with scrolling camera
- Lives system (5 lives) and score tracking
- Collectible purple gems/stars (10 total)
- Enemy hazards with varied movement patterns
- Level completion goal (flag and castle)
- Death and respawn mechanics
- Restart functionality
