# User Game Preferences - Super Kiro World

## Technical Stack
- Vanilla JavaScript, HTML5 Canvas, CSS
- Single HTML file that opens directly in browser

## Character & Visuals
- Character: Kiro-logo.png with directional facing
- Collectibles: Purple gems/stars matching Kiro brand (#790ECB)
- Color scheme: Kiro purple colors for platforms and UI
- Dark background with brand colors

## Game Mechanics
- Physics: gravity 0.5, jumpPower 12, speed 5, friction enabled
- Lives: 5 starting lives
- Scoring: 1 point per collectible
- Lose life when: falling off screen or touching enemies

## Level Design
- 10 platforms with gaps
- Include hazards (pits and enemies)
- Scrolling camera following player
- Platform-based level design

## Gameplay Features
- Move left/right with arrow keys
- Jump with proper physics
- Collision detection for platforms, collectibles, enemies
- Score counter and lives display
- Level completion and restart functionality
